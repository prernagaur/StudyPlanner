# StudyPlanner
The Personalised Study Planner allows users to create and manage customized study plans based on their subjects and available time. It helps improve time management, consistency, and productivity.

Highlights :
1. Create personalized study schedules
2. Add subjects and study tasks
3. Track study progress
4. Simple and user-friendly interface

Tech Stack Used :
1. HTML
2. CSS
3. JavaScript

Purpose of the Project :
1. Improve time management
2. Organize study routines
3. Reduce study-related stress
4. Apply web development concepts
